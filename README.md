# AI Task Manager

AI-powered task management CLI tool built with TypeScript and Node.js.

## Usage

### Initialize a workspace
```bash
npx @e0ipso/ai-task-manager init --no-user-interaction
```

## Development

### Build
```bash
npm run build
```

### Test
```bash
npm test
```

### Lint and Format
```bash
npm run lint
npm run format
```
